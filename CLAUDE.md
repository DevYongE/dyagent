# dyagent 플러그인 저장소

이 저장소는 Claude Code 플러그인 **dyagent**(회사형 멀티에이전트 개발 하네스)의 소스다.

## 하네스: dyagent

**목표:** 대표(메인 세션)가 직원 에이전트 12명을 지휘해 요구사항 회의 → 조사 → 기획 → 디자인 → 병렬 구현 → 검증(내부 QA + 외주 Codex) → 옵시디언 납품 → 마케팅(스크린샷·블로그·인스타·유튜브)까지 수행.

**트리거:** 이 저장소에서의 작업은 플러그인 자체의 수정·개선이다. 하네스 실행은 플러그인을 설치한 프로젝트에서 `dyagent` 스킬로 트리거된다.

**구조:** `agents/` 직원 정의 12개, `skills/dyagent/SKILL.md` 오케스트레이터, `.claude-plugin/` 매니페스트(plugin.json + marketplace.json).
